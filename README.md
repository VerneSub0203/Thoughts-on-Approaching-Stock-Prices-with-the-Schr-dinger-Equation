シュレディンガー方程式のアプローチを株価分析に転用し、市場の隠された構成要素を抽出する次世代クオンツモデルの検証リポジトリです 。物理学の閉鎖系と異なり、金融市場は予測不能な外部ノイズの干渉を受ける開放系です 。本プロジェクトでは、生データから状態を解剖し、ノイズフィルタリング等を用いて外部からの力を補正することで、新たな経済的洞察の獲得を目指します 。

# Open Quantum System Dynamics

量子力学において、システムが外部の環境とエネルギーや粒子、情報をやり取りできる「開放系（Open Quantum System）」のダイナミクスを記述・シミュレーションするためのリポジトリです。

閉鎖系における「エネルギー保存則」と「全確率の保存（エルミート性）」という前提を取り払い、環境への「散逸」や「ノイズの流入」を数学的に表現するため、方程式の構造を大きく2つの方向性で拡張してアプローチしています。

---

## 1. 非エルミート・シュレディンガー方程式

最も直感的な拡張として、ハミルトニアン $\hat{H}$ を非エルミート化する手法です。環境へ粒子やエネルギーが一方的に逃げていく散逸系をモデル化するため、エネルギーの固有値 $E$ に虚数項を導入します。

### 定義式
$$\hat{H} = \hat{H}_0 - i\Gamma$$

* $\hat{H}_0$: 元のエルミートなハミルトニアン
* $\Gamma$: 環境への散逸力を表す正の定数演算子

### 時間発展
これを通常のシュレディンガー方程式 $i\hbar \frac{\partial}{\partial t}|\psi(t)\rangle = \hat{H}|\psi(t)\rangle$ に代入して解くと、時間発展は以下のようになります。

$$|\psi(t)\rangle = e^{-\frac{i}{\hbar}\hat{H}_0 t} e^{-\frac{\Gamma}{\hbar} t} |\psi(0)\rangle$$

* **第一項** $e^{-\frac{i}{\hbar}\hat{H}_0 t}$: 通常の量子的な「振動（位相の変化）」
* **第二項** $e^{-\frac{\Gamma}{\hbar} t}$: 振幅が指数関数的に小さくなる「減衰」

波動関数の絶対値の2乗（存在確率）は時間とともに $e^{-\frac{2\Gamma}{\hbar}t}$ で減少します。この「確率が保存されない（100%から減っていく）」性質が、外部へ漏れ出している開放系の状態を記述します。

## 2. リンブラッド・マスター方程式

散逸だけでなく、環境からのランダムなノイズ（キック）や量子的なデコヒーレンス（重ね合わせの崩壊）までを扱う場合、システムは「純粋状態」から「混合状態」へ移行するため、状態の記述を波動関数 $\Psi$ から密度行列 $\rho$ へと切り替えます。

### Lindblad Master Equation
閉鎖系のフォン・ノイマン方程式に、環境との相互作用による効果を追加した開放系量子力学の金字塔とも言える方程式です。

$$\frac{\partial \rho}{\partial t} = -\frac{i}{\hbar}[\hat{H}, \rho] + \sum_k \left( L_k \rho L_k^\dagger - \frac{1}{2} \{ L_k^\dagger L_k, \rho \} \right)$$

* **ユニタリ項** $-\frac{i}{\hbar}[\hat{H}, \rho]$: 
  従来のシュレディンガー方程式に対応するエルミートな時間発展部分。
* **散逸項** $\sum_k \dots$: 
  「リンブラッド散逸項」。$L_k$ は環境へのジャンプを記述する散逸演算子。この項により、全確率を $1$ に保ったまま、量子的な干渉性が失われていく様子や熱平衡状態への緩和過程を計算可能になります。
